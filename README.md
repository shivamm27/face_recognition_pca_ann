## 🎯 **Aim**

The aim of this project is to develop an **end-to-end face recognition system** using **Principal Component Analysis (PCA)** for dimensionality reduction and **Artificial Neural Networks (ANN)** for classification, thereby achieving efficient and accurate recognition of human faces.

---

## ✅ **Objectives**

1. **Data Collection & Preprocessing**

   * Gather face images of different persons and preprocess them (grayscale conversion, resizing, normalization).

2. **Dimensionality Reduction using PCA**

   * Apply PCA to extract **eigenfaces** and reduce the high-dimensional image data into a lower-dimensional subspace.

3. **Feature Representation**

   * Represent each face image in terms of principal components (eigenfaces) to capture the most significant variations.

4. **Model Training with ANN**

   * Train an **Artificial Neural Network classifier** on the reduced feature set obtained from PCA.

5. **Performance Evaluation**

   * Test the trained model on unseen face images and evaluate accuracy, loss, and classification performance.

6. **Visualization & Results**

   * Visualize the **mean face**, **eigenfaces**, and training performance graphs.
   * Save and deploy the trained ANN model for predictions on new input images.

---

## 🏆 **Conclusion**

This project successfully demonstrates the application of **PCA and ANN** in face recognition.

* PCA reduced the dimensionality of the input data while preserving essential features.
* ANN provided a powerful classification mechanism, learning to distinguish between different faces effectively.
* The system achieved promising accuracy (e.g., \~90% in your case) on the dataset.

Thus, the combination of PCA and ANN proves to be a robust approach for building efficient face recognition systems, balancing computational efficiency with recognition accuracy.
